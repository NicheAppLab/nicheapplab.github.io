Windows 用の T-Code IME のテストリリースができました([Link](https://github.com/NicheAppLab/T-Code-IME-for-Win/releases/tag/0.1.0-RC1)).

## 注意点
- JRE(or JDK)が必要で, PATH/JAVA_HOME の設定が必要. 概ねインストーラーが設定してくれますが, 設定されない環境もあるかもしれません.
- IME はキーボードとして追加が必要です. コントロールパネルより, 言語と地域から, 日本語の設定で, 下方にキーボードの設定があります. そこに追加する必要があります.
- IME を動かすにはインストールされた T-Code IME Proxy を起動する必要があります.
- T-Code モード以外は未サポート (半角英字にするときは IME の切り替えでいいかな,と…)
- 問題があれば私に DM か GitHub にて [Issues](https://github.com/NicheAppLab/T-Code-IME-for-Win/issues) に登録していただけると幸いです.
